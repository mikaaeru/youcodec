# YouCodec
**by michellesw v1.0**

A minimal YouTube extension to change streaming codecs, toggle HDR, and audio track selection.

## Features
* **Codec Selection**: Choose between AV1, VP9, or H.264.
* **HDR Toggle**: Enable or disable High Dynamic Range content globally.
* **Force Original Audio**: Automatically switches from auto-dubbed tracks back to the creator's original audio.

## Installation
1. Clone or download this repository.
2. Open Chrome/Edge and navigate to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select the project folder.
